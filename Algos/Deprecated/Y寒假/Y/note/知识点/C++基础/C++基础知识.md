# C++基础知识



#### 迭代器访问

在 C++ 中，迭代器是一种类似于指针的对象，用于遍历容器（如 `std::unordered_map`、`std::vector` 等）中的元素。

两种访问方法

```c++
cout << *it.first << " " << *it.second << endl; // 使用 . 访问成员
```

`*` 是解引用运算符， 解引用之后就可以直接获得

或者

```C++
cout << it->first << " " << it->second << endl;
```

`->` 是 C++ 中的成员访问运算符，用于通过 **指针** 或 **迭代器** 访问对象的成员。
